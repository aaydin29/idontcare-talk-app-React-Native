## I do not care! Talk App / React Native 
- In this project, I made an application where you can send messages by logging in, and the messages you send can be seen by everyone.
- I used **Firebase** while developing the app.
- I used firebase on **auth** module for registration, login and logout operations.
- I used the **database** module to control and use the data in the application.
- I also used the **FlashMessage** build to be a nice display for error messages.
- I also used the **date-nfs** to show the time in the messages better.
